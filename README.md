# AdminSetup Package For Laravel 5.5

__Web interface for help with deploying of application__<br>


### Database
* Drop all tables


### Migrations
* Run migrations


### Seeding
* Seed all tables
* Seed specific table
* Generate seeds


### Install package
``` bash
composer require vilbur/admin-setup @dev
```

### Interface view
``` html
http://your-domain/admin-setup
```
